Colour Mix.
Try to mix the target colour!
GitHub link
  • Code
Description
A simple game where the player has the select values of red, green and blue to be mixed together to create the target colour. There are three rounds and the score from each round is combined for a total score. The score is calculated by summing the difference between the user selected and actual values for the red, green and blue to get a total difference, then dividing that by the total possible difference (255*3) and muliplying by 100 to get a percentage. Then 100 - that percentage to get a score out of 100.
Technical Description
The app is entirely built inside React using typescript and was largely created as a learning opportunity for me to build my React skills as I had previously largely used Angular
